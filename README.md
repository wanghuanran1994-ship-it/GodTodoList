# GodTodoList

全栈事项管理体系 — 目标驱动的高效任务管理。

## 一键启动

```bash
npm install && npm start
```

浏览器打开 `http://localhost:3000`。

## 功能

- **看板视图** — 3 列拖拽管理（正在进行 / 搁置 / 已完成）
- **工作台** — 今日任务、即将到期、最近完成概览
- **日历** — 周/月视图规划
- **目标管理** — 目标 → 任务关联，进度追踪
- **惯例系统** — 周期性任务模板（周报、月报等）
- **汇报管理** — 按会议分类管理汇报任务
- **AI 增强** — 自然语言快速创建、AI 分解子任务、AI 分析进展
- **AI 会话关联** — 关联 Claude Code / OpenCode 对话到任务
- **计时器** — 任务级番茄钟
- **批量操作** — 多选批量改状态、改目标、删任务
- **文件管理** — 拖拽上传、自动建目录
- **笔记卡片** — 便签式笔记系统
- **深色模式** — 支持亮/暗切换
- **数据备份** — 一键下载/恢复数据库

## 技术栈

- 前端：Vue 3 (CDN) + 原生 CSS
- 后端：Express + sql.js (SQLite via WASM)
- 数据库：SQLite，单文件 `data/godtodo.db`
